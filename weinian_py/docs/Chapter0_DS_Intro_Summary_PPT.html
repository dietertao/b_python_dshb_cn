NumPy核心知识框架

核心理念：数据结构 + 操作

核心数据结构：ndarray (N维数组)

  • 1维数组 = 向量 (Vectors)
  • 2维数组 = 矩阵 (Matrices)
  • 多维数组 = 张量 (Tensors)

3大核心特征：

  • 🧱同质性： 所有元素必须同类型
  • 📐维度固定： 创建时确定形状
  • 💾连续内存： 元素物理存储连续

快速自检方法：

  • array.shape - 维度结构
  • array.dtype - 元素类型
  • array.ndim - 维度数
  • array.size - 元素总数

NumPy四大操作原则

高效数据操作法则

原则1：向量化优先 (避免循环)

# ❌ Python循环方式 squares = [x**2 for x in range(1000)] # ✅ NumPy向量化方式 arr = np.arange(1000) squares = arr ** 2 # 速度提升50倍以上

其他三个原则：

  • 原则2：广播机制 - 从后往前对齐维度，缺失维度视为1
  • 原则3：视图而非复制 - view = arr[:2, :] vs copy = arr.copy()
  • 原则4：轴(axis)方向操作 - axis=0(跨行), axis=1(跨列)
